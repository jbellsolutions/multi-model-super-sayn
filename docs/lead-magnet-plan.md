# Lead Magnet Plan

## Offer Framing

This should be packaged as a free, high-value starter system for people who want to learn how to build with a Claude Code ecosystem and multi-agent orchestration.

Primary hook:

**Build your own AI agent team locally.**

Secondary hook:

**Use the repo as the entry point into Claude Code Ecosystem Certification from AI Integrators.**

## Who This Is For

- consultants building AI delivery systems
- technical founders
- internal AI leads
- operators who want an orchestrator without starting from scratch

## What The Free Repo Should Include

- local dashboard
- orchestrator starter framework
- agent role templates
- setup flow for provider auth
- example prompts
- example orchestration plans
- cost/savings explainer
- installation walkthrough
- short use-case library

## What The Landing Experience Should Communicate

Within one screen, the user should understand:

- what this repo is
- why orchestration beats single-model chat
- what kinds of outcomes it can help build
- how fast they can get started
- what certification or next-step offer exists

## Copy Direction

The tone should feel:

- technical
- credible
- high-energy
- practical

Avoid:

- vague "AI magic" language
- overclaiming autonomy
- anime references that imply franchise affiliation

## Brand and Visual Direction

Use an original "power-up" design language:

- gold, ember, obsidian, steel, electric cyan accents
- kinetic lines
- aura gradients
- sharp silhouettes
- tactical dashboard UI

Do not use copyrighted franchise art, names, symbols, or character lookalikes.

## Repo README Direction

The root README should be rewritten around:

1. What you can build
2. How the orchestration works
3. What you see in the dashboard
4. How to run it locally
5. Why it saves money
6. Where to go next for certification
