# Use Cases

## 1. Build A SaaS MVP From One Brief

### Prompt

"Build a SaaS MVP for appointment booking with a marketing site, login flow, dashboard, Stripe integration, and onboarding emails."

### Recommended team

- Orchestrator
- Gemini Researcher
- Codex Worker x2
- Multi Reviewer

### Why it works

- researcher defines stack and feature trade-offs
- coding agents split frontend and backend work
- reviewer checks production risk before wrap-up

### Good video angle

Show the planner converting one vague request into a staged build plan with specialist agents.
